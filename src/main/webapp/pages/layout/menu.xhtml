Showcase menu This page desmonstrates showcase menu and top bar code

The menu is defined via facelets templating by *extending* Admin Template.

Note

Menu itens are activated (class active) based on current url and menu href.

       <ui:include src="/includes/menu-search.xhtml"/>
  </ui:define>
  <ui:define name="sidebar">
       <ui:include src="/includes/menu.xhtml"/>
  </ui:define>

  <ui:define name="topMenu">
       <ui:include src="/includes/top-menu.xhtml"/>
  </ui:define>

</ui:composition>
